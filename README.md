# SavvyCal Clone (React + Tailwind CSS)

A modern landing page clone built with React and Tailwind CSS, featuring a responsive design and interactive components.

## 🚀 Features

-   Responsive navigation bar with mobile support
-   Interactive hero section with testimonials
-   Custom animations and hover effects
-   Modern UI components
-   Custom font integration
-   Tailwind CSS styling

## 🛠️ Technologies

-   React
-   Tailwind CSS
-   Vite
-   ESLint
-   SWC

## 🌐 Project Structure

```
src/
├── assets/ # Static assets
├── components/ # React components
│ ├── HeaderTestimonial.jsx
│ ├── Hero.jsx
│ ├── LinkButton.jsx
│ └── Navbar.jsx
├── App.jsx # Main App component
├── index.css # Global styles
└── main.jsx # Entry point
```

## 📦 Installation

1. Clone the repository:
    ```bash
     git clone https://github.com/rudradcruze/tailwindProjectLearn.git
    ```
2. Navigate to the project directory:
    ```bash
    cd tailwindProjectLearn
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev
    ```
